Game Projects

Gemma's Great Gambit - Made in Unity
January - May 2017
Programmer
Gemma's Great Gambit was created with a team composed of 3 programmers, a sound designer, an artist and a writer. It is a 2D puzzle platformer game where the protagonist must use an ancient relic to open up portals to alternate dimensions and save the day!
Break Up Simulator - Made in Unity
January 2015
Programmer/Writer
Break Up Simulator is a take on the dating sim genre where the goal is to manage to break out of your arranged marriage. It figures branching dialogues and different endings based on player choices. The GGJ theme was "What do we do now?" To make this project more challenging a conversation system was created to dynamically allow the writers to add images and dialogue.
Cat Effect - Made in Unity
January 2014
Programmer
Cat Affect is a 2D puzzle platformer where you play as Alex the cat. The theme for this GGJ was "We don't see things as they are, we see them as we are.". We chose to interpret this as: things that happen to your Alex the cat happen to the whole world. So when Alex gets wet, the world is wet, when Alex jumps in a fire, well... Enjoy! This game was the first crack at Unity for the team, so it was used as a learning experience, some things.
